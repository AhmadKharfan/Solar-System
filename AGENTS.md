# Project Instructions

This is an Android Jetpack Compose project.

The app should prioritize:
- Figma fidelity,
- clean Compose architecture,
- GPU-friendly animation,
- low jank,
- low overdraw,
- Material 3 consistency,
- performance measurement.

When implementing Figma designs:
- Use Figma MCP.
- Use exact frame/layer links.
- Prefer existing components.
- Use MaterialTheme and design tokens.
- Implement motion as state-driven Compose animation.
- Prefer graphicsLayer for transform-based animation.
- Avoid layout-heavy animation unless required by the design.

Performance must be verified with real tools, not guessed.
